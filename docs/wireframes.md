# 画面遷移図とワイヤーフレーム

## 1. 画面遷移図

```mermaid
graph TD
  %% 共通画面
  Login[ログイン画面]
  Profile[プロフィール編集画面]
  Logout[ログアウト]

  %% 会員用画面
  Dashboard[ダッシュボード]
  Reserve[コース予約]
  Manage[予約管理]
  CheckIn[入館登録]
  Absence[欠席登録]
  Plan[プラン管理]

  %% 管理者／コーチ用画面
  AdminDashboard[管理者ダッシュボード]
  CourseMgmt[コース管理]
  CoachMgmt[コーチ管理]
  CourtMgmt[コート管理]
  MemberMgmt[会員管理]
  Reports[レポート]

  %% 遷移
  Login --> Dashboard
  Dashboard --> Reserve
  Dashboard --> Manage
  Dashboard --> CheckIn
  Dashboard --> Absence
  Dashboard --> Plan
  Dashboard --> Logout
  Dashboard --> Profile

  Reserve --> Manage
  Manage --> Absence
  Manage --> CheckIn

  %% 管理者フロー
  Login --> AdminDashboard
  AdminDashboard --> CourseMgmt
  AdminDashboard --> CoachMgmt
  AdminDashboard --> CourtMgmt
  AdminDashboard --> MemberMgmt
  AdminDashboard --> Reports
```

## 2. ワイヤーフレーム

### 2.1 ログイン画面
```
+------------------------+
|        テニス 予約管理       |
+------------------------+
| ユーザー名 [__________]   |
| パスワード [__________]   |
| [ ログイン ] [パスワード再発行] |
+------------------------+
```

### 2.2 ダッシュボード画面（会員用）
```
+------------------------------------------------+
| ダッシュボード                                  |
+------------------------------------------------+
| [入館登録]                                     |
| 残予約枠：2 [コース予約]                        |
| 今週のレッスン一覧                             |
|  - 4/1 18:00 初級 (コーチA, コート1) [欠席登録] |
|  - 4/3 20:00 中級 (コーチB, コート2) [欠席登録] |
+------------------------------------------------+
| [予約管理] [プラン管理]                         |
+------------------------------------------------+
```

### 2.3 コース予約画面
```
+------------------------------------------------+
| コース予約                                    |
+------------------------------------------------+
| レベル: [ 初級 ▼ ]  日付: [ 4/ ] ▼  時間: [ ▼ ]|

| | カレンダー表示 (空き日を色付)         |               |
| +----------------------------+               |
| [ 検索 ]                                    |
| 検索結果                                     |
|  - 4/8 18:00 初級 (コーチA, コート1) [予約] |
+------------------------------------------------+
```
